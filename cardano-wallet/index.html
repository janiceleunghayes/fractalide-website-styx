Cardano Wallet

What is Cantor?

Cantor is a Cardano Wallet and is Fractalide's first application. The wallet allows you to send and receive ADA transactions. Please consider this alpha software - not to be used in anger.

Download Cantor

Cantor features

Built using reusable, reproducible and composable software components

This design pattern allow people to easily reuse components to build new wallets to support other cryptocurrencies. Each component is a well specified module that composes. A composition of well specified components used to implement a top level specification helps reduce the semantic gap considerably.

Adheres to the formal
specification of a Cardano Wallet

Cantor adheres to the Cardano Wallet Formal Specification document. This allows for a better implementation that accurately maps reality. This reduces architectural problems. Upgrading Cardano becomes smoother and more predictable.

Designed for long-term integration
with Fractalide DApps

Hyperflow is designed to be a browser for descentralized applications. Other dapps will most likely need to intereact with Cardano's currency ADA. All dapps need to do is call in Cantor and the functionality becomes available.
